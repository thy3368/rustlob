# Brand kit

The page contains downloadable brand assets for Hyperliquid:

- **Hyperliquid logo (PNG format)** – Logo files in PNG format
- **Hyperliquid logo (SVG format)** – Logo files in vector SVG format
- **Hyperliquid banner** – Banner image asset
- **Hypurr** – Branded mascot/character asset (version 1)
- **Hypurr 2** – Second version of the mascot asset
- **Hypurr 3** – Third version of the mascot asset
- **EVM, Core, Powered by Hyperliquid** – Co-branded asset for EVM and Core integrations

All assets are available as downloadable files from the Hyperliquid documentation repository.
