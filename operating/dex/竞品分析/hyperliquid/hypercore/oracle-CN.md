# 预言机

验证者每 3 秒为每个永续资产发布现货预言机价格。这些价格决定资金费率并贡献于标记价格，影响保证金、清算和 TP/SL 订单触发。

现货预言机价格计算为"Binance、OKX、Bybit、Kraken、Kucoin、Gate IO、MEXC 和 Hyperliquid 现货中间价的加权中位数"，权重分别为 3、2、2、1、1、1、1、1。Hyperliquid 上主要现货流动性的资产（如 HYPE）在足够流动性发展之前排除外部来源。主要流动性在其他地方的资产（如 BTC）从预言机计算中排除 Hyperliquid 现货价格。

清算所对所有验证者提交进行最终加权中位数计算，验证者根据其质押权重加权。
