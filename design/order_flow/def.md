Order Flow 交易常用指标统计方式

以下是订单流交易中常用指标的统计方法和计算逻辑：

一、核心指标统计方法

1. Delta（订单流不平衡）

统计方式：

Delta = 主动买入成交量 - 主动卖出成交量
累积Delta = ∑(每笔Delta)


计算要点：
• 主动方判断：通过成交价与买卖一档比较

• 成交价 ≥ 卖一价 → 主动买入

• 成交价 ≤ 买一价 → 主动卖出

• 时间窗口Delta：按固定时间窗口（如1秒、1分钟）统计

• 价格区间Delta：按价格区间统计，识别关键价位的买卖压力

2. 成交量分布（Volume Profile）

统计方式：

1. 按价格区间统计总成交量
2. 计算每个价格区间的成交量占比
3. 识别关键水平：
    - POC（Point of Control）：成交量最大的价格
    - VAH/VAL（Value Area High/Low）：累计成交量70%的上下边界


计算步骤：
1. 将价格划分为固定间隔（如Tick大小或自定义区间）
2. 统计每个价格区间的总成交量
3. 按成交量排序，找出POC
4. 从POC向两侧扩展，直到累计成交量达到总成交量的70%

3. 订单簿不平衡度

统计方式：

买卖盘不平衡度 = (买盘总量 - 卖盘总量) / (买盘总量 + 卖盘总量)


多档不平衡统计：

前N档不平衡度 = (∑前N档买量 - ∑前N档卖量) / (∑前N档买量 + ∑前N档卖量)


4. 大单追踪

统计方式：

1. 设定大单阈值（如：> 平均成交量的5倍）
2. 统计大单出现频率
3. 分析大单成交方向（主动买入/卖出）
4. 追踪大单吸收情况


大单吸收率计算：

吸收率 = 大单后价格回归幅度 / 大单冲击幅度


5. 足迹图（Footprint）指标

统计方式：

1. 按价格和时间维度统计成交量
2. 区分主动买入和主动卖出成交量
3. 计算每个价格水平的买卖比例


关键模式统计：
• 单边拍卖：某价格水平买卖比例 > 80%

• 停止单：价格快速穿透关键水平后反转

• 流动性缺口：连续价格无成交

二、高级指标统计方法

6. CVD（累积成交量Delta）

统计方式：

CVD_t = CVD_{t-1} + Delta_t


分析要点：
• CVD斜率变化率

• CVD与价格的背离

• CVD在关键水平的支撑/阻力

7. TPO（时间价格机会）分析

统计方式：

1. 按固定时间区间（如30分钟）划分
2. 统计每个价格在时间段内出现的频率
3. 构建TPO图，识别市场轮廓


8. 吸收指标

统计方式：

吸收强度 = 吸收期间成交量 / 冲击订单量
吸收效率 = 价格回归幅度 / 冲击幅度


吸收类型判断：
1. 有效吸收：大单后价格快速回归，成交量放大
2. 无效吸收：大单后价格持续下跌，成交量萎缩

9. 流动性指标

统计方式：

市场深度 = ∑(前N档买卖挂单量)
紧密度 = 买卖价差 / 中间价
弹性 = 价格偏离后回归的速度


三、统计时间框架

高频统计（Tick级）

• 每笔成交的Delta计算

• 实时订单簿不平衡

• 大单即时检测

短期统计（秒/分钟级）

• 累积Delta变化

• 成交量分布更新

• 吸收行为分析

中期统计（小时/日级）

• CVD趋势分析

• 关键水平识别

• 市场结构分析

四、数据预处理要求

数据清洗

1. 异常值过滤：
   • 过滤零成交量成交

   • 过滤价格跳跃过大成交

   • 过滤非交易时段数据

2. 数据对齐：
   • 成交数据与订单簿数据时间对齐

   • 不同数据源时间同步

数据增强

1. 主动方向标记：
   • 比较成交价与买卖一档

   • 考虑成交时间序列

2. 大单识别：
   • 动态阈值调整

   • 考虑市场流动性

五、统计结果可视化

实时监控指标

1. Delta热力图：按价格和时间显示Delta强度
2. 成交量分布图：显示POC和VAH/VAL
3. 订单簿深度图：显示买卖盘堆积
4. 足迹图：显示每个价格水平的详细成交

历史分析图表

1. CVD与价格对比图
2. 大单分布散点图
3. 吸收行为统计图
4. 市场轮廓图（TPO）

六、统计优化建议

计算效率优化

1. 增量计算：避免全量重算
2. 滑动窗口：实时更新统计结果
3. 近似计算：对非关键指标使用近似算法
4. 并行计算：多指标并行统计

内存管理

1. 数据分片：按时间或品种分片存储
2. 缓存策略：热点数据内存缓存
3. 压缩存储：历史数据压缩存储

七、实战统计示例

示例1：日内Delta统计


统计周期：5分钟
计算步骤：
1. 每5分钟重置Delta统计
2. 计算该周期内：
    - 总Delta值
    - 最大单笔Delta
    - Delta正负比例
    - Delta与价格相关性


示例2：关键价位统计


统计目标：识别支撑阻力位
计算方法：
1. 统计历史高成交量区域
2. 计算价格在这些区域的反应：
    - 反弹次数
    - 突破概率
    - 停留时间


示例3：大单行为统计


统计维度：
1. 大单出现时间分布
2. 大单成交方向统计
3. 大单后价格走势：
    - 1秒内价格变化
    - 10秒内价格变化
    - 1分钟内价格变化


八、注意事项

数据质量

1. 数据完整性：确保成交和订单簿数据完整
2. 时间精度：毫秒级时间戳精度
3. 数据源一致性：避免多个数据源冲突

统计偏差

1. 幸存者偏差：仅分析成功交易
2. 过拟合风险：在历史数据上过度优化
3. 市场变化：统计规律随时间变化

实际应用

1. 多市场验证：在不同市场验证统计规律
2. 参数敏感性：测试不同参数下的稳定性
3. 实时监控：建立实时监控和预警系统

总结

订单流指标的统计需要结合市场微观结构理论，通过科学的统计方法从海量数据中提取有效信号。建议：

1. 从简到繁：先掌握基础指标统计，再逐步增加复杂度
2. 验证回测：所有统计方法都要经过历史数据验证
3. 实盘测试：小资金实盘测试统计有效性
4. 持续优化：根据市场变化调整统计方法

有效的订单流统计不仅能提供交易信号，更能帮助理解市场运行机制，是量化交易的重要基础。
