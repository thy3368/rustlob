# API

Python SDK: https://github.com/hyperliquid-dex/hyperliquid-python-sdk

Rust SDK（由社区成员 Infinite Field 编写）: https://github.com/infinitefield/hypersdk

Typescript SDKs（由社区成员编写）:
- https://github.com/nktkas/hyperliquid
- https://github.com/nomeida/hyperliquid

CCXT 还维护符合标准 CCXT API 的多语言集成: https://docs.ccxt.com/#/exchanges/hyperliquid

所有 API 调用示例使用主网 URL (https://api.hyperliquid.xyz)，但您可以使用相应的测试网 URL (https://api.hyperliquid-testnet.xyz) 发出相同的请求
