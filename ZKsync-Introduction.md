# ZKsync 介绍文档

## 概述

**ZKsync** 是一个面向以太坊的第二层（Layer 2）解决方案，也是业界首个**EVM兼容的零知识证明 Rollup**（ZK Rollup）。它使用零知识密码学技术大幅提升以太坊的交易吞吐量和处理速度，同时维持与以太坊网络的安全性。

## 主要用途

### 1. **提升以太坊扩展性**
- **增加吞吐量**：通过链下批处理多笔交易，然后生成零知识证明提交至以太坊，实现吞吐量从约15 TPS 提升到数千 TPS
- **降低交易费用**：相比以太坊主网，交易成本大幅下降（通常便宜 100-200 倍）
- **加快确认速度**：交易几秒内完成，而无需等待以太坊区块确认

### 2. **保持以太坊兼容性**
ZKsync Era 实现了 **EVM 兼容**（Ethereum Virtual Machine Compatible），这意味着：
- 现有 Solidity 合约可直接部署，无需重新编写
- 开发者可使用熟悉的 Ethereum 工具链（Hardhat、Truffle 等）
- Web3 钱包和应用无缝支持

### 3. **DeFi 应用场景**
- **去中心化交易所（DEX）**：高频、低成本的交易配对
- **借贷协议**：快速清算和头寸管理
- **资产交换**：跨资产交易的高效执行

### 4. **支付和转账**
- **小额支付**：交易费用极低，适合大规模支付场景
- **跨链桥接**：与以太坊的双向资产转移
- **国际汇款**：低成本跨境转账

## 核心技术：zkEVM

### 什么是 zkEVM？

**zkEVM**（零知识以太坊虚拟机）是 ZKsync 的核心创新，一个能够生成零知识证明的虚拟机：

- **交易执行**：与标准 EVM 兼容，执行智能合约
- **证明生成**：在保证正确性的同时生成零知识证明
- **状态验证**：无需披露交易细节，即可证明新状态的正确性

### 工作原理

```
原始区块链状态
    ↓
处理多笔交易（链下）
    ↓
输出新状态 + 零知识证明
    ↓
提交证明到以太坊
    ↓
以太坊验证证明 → 确认新状态有效
```

## ZKsync 的关键特性

### 1. **安全性**
- 所有交易通过零知识证明验证
- 继承以太坊的安全性，无需信任中心化验证者
- 不依赖任何保管人或运营者

### 2. **高效性**
- 优化的 zkEVM 架构实现快速证明生成
- 自定义编译器支持高性能执行
- 批量处理多笔交易，分摊证明成本

### 3. **开发友好**
- Solidity 合约支持
- 兼容现有 Ethereum 工具和库
- 完整的开发者文档和 SDK

### 4. **互操作性**
- 与以太坊主网的双向桥接
- 支持 ERC-20、ERC-721 等标准代币
- 可与其他 Layer 2 方案跨链交互
