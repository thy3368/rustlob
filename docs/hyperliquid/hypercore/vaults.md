# Vaults

Vaults represent a general-purpose implementation of HyperEVM's powerful capabilities through CoreWriter and precompiles. Developers can establish and tokenize vaults on HyperEVM with complete customization of accounting logic. They may adhere to standards like EIP-4626 while gaining the advantage of "trustless read and write operations on HyperCore." For instance, vaults can execute onchain trades via CoreWriter or authorize multiple agents using the same mechanism. This delivers fully onchain accounting through precompiles and complete access to Core features including spot and HIP-3 across all quote assets.

This approach significantly improves upon the original HyperCore vaults from 2023, which lacked HIP-3 support and spot trading functionality.
